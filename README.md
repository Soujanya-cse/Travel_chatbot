# 🧳 Travel Assistance Chatbot  

The **Travel Assistance Chatbot** is an AI-powered assistant that helps users with essential travel details such as **weather updates**, **hotel suggestions**, and **flight information**.  
This project is built using the **Google Gemini API** for conversational AI.  

---

## 🚀 Features  
- 🌦️ **Weather Info** – Get real-time weather details for your destination  
- 🏨 **Hotel Suggestions** – Find suitable accommodation options  
- ✈️ **Flight Details** – Explore available flight options  

---

## 🛠️ Tech Stack  
- **Language:** Python  
- **API:** Google Gemini API  
- **Interface:** Command Line / Console
